Приложение проекта SberBankCredit состоит из двух частей
1) SchesulePaymentsGPPTest - демонстрирует, имплементацию класса по расчету графика плановых платежей, с выводом графика в Excel,
с помощью биьлиотеки ApachePoi
На входе есть  файл input \src\main\resources\input\input.txt
В нем задаются следующие входные параметры
amount:100000
percent:10
date:28.05.2018
term:36

На выходе имеем файл C:\Users\Евгений\IdeaProjects\SberBankCredit\src\main\resources\output\payments.xml
с расчитанным графиком (формула аннуитетного графика)

В приложении реализованы следующие методы

checkInputParams() - проверка входных данных по заданным условиям и выход из приложения в случае не соблюдения условий.
checkInputParamsAndGetDefault() - расширяет функционал checkInputParams, в случае не соблюдения условий ,
возвращает дефолтные значения из классов ValidateException
calculate() - метод расчета  ГПП
sorting() - сортировка списка платежей, по дате платежа(реализовано в виде анонимных классов)
buildExcel() - вывод графика плановых платежей в Excel

2)
Реализация  связки JDBC+H2
TransferTest - демонстрирует , операцию перевода денежных средств с одного счета на другой(алгоритм для разных счетов)
метод transferSingleThreadTest - в однопоточной среде
метод теста transferThreadsTest - запускает несколько потоков


jdbc\entity\AccountDTO - реализация Entity слоя
jdbc\dao\TAccountDAO - интерфейс Dao слоя для Сервиса
jdbc\service\AccountService - реализация service слоя - класс запросов с возможностью добавления нового счета,
 удаления, начитки списка счетов, и перевода средст с одного счета на другой
exception\TransferException - созданное исключение, выбрасывающееся в случае не хватки средст на счете источнике, или
в случае если операция проходит между одним счетом.
jdbc\impl\TransferOperation - основной класс трансфера между счетами с методом transferAmount

На вход нам приходят два номера счета источника и приемника, а так же сумма перевода
Далее мы по номерам начитываем данные из таблицы и создаем объекты Account
Далее запускаем процесс перевода средств.
